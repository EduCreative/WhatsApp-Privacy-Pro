# WhatsApp Privacy Pro 🛡️

Advanced privacy toolkit for WhatsApp Web. Take control of your digital boundaries with professional-grade privacy features.

## 🚀 Features

- **Selective Blur**: Blur messages, photos, names, and previews independently.
- **Panic Mode**: Instantly hide your chats with a customizable hotkey (`Alt + Q`).
- **Incognito Mode**: Hide blue ticks and "typing..." indicators at the network level.
- **Custom Intensity**: Adjust the blur strength to your preference.
- **Privacy First**: All processing happens locally in your browser. No data is ever sent to external servers.

## 🛠️ Installation

1. **Download**: Download the source code as a ZIP file and extract it.
2. **Developer Mode**: Open Chrome and go to `chrome://extensions`.
3. **Enable**: Toggle "Developer mode" in the top right corner.
4. **Load**: Click "Load unpacked" and select the extracted folder.

## 💻 Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: GitHub Actions
